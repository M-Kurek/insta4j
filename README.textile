h1. Welcome to insta4j

The java client api for Instapaper "Full":http://www.instapaper.com/api/full and "Simple":http://www.instapaper.com/api/simple
API which allows third-party applications to add URLs to Instapaper.

h2. Getting started

Fork the project source code on "Github":https://github.com/dzontak/insta4j

bc. git clone git@github.com:dzontak/insta4j.git

1. Build the client with maven
mvn install

2. Add insta4j-client and insta4j-simpleclient as dependencies to your pom.

<dependency>
  <groupId>com.idtmatter</groupId>
  <artifactId>insta4j-client</artifactId>
  <version>1.0.0</version>
</dependency>

<dependency>
  <groupId>com.idtmatter</groupId>
  <artifactId>insta4j-simpleclient</artifactId>
  <version>1.0.0</version>
</dependency>
